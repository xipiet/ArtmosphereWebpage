# Artmosphere - Orakel

- Web-App mit /ipad und /main <br/>

## Installation

- git clone <br/>
- cd ArtmosphereWebpage <br/>
- Node & npm installieren (https://nodejs.org/en/download) <br/>
- npm init -y <br/>
- npm install <br/>
- npm install ogl <br/>
- npm install gsap <br/>
(sollte die benutzten packages automatisch installieren, ansonsten manuell)  <br/>
- npm run dev <br/>

## Update

- ps aux | grep vite <br/>
- kill "entsprechende PID" <br/>
- cd /ArtmosphereWebpage <br/>
- git pull <br/>
- reboot <br/>
